# Computer Components Store

## Web store which provides possibility to buy computer components

### Entities

* User(Client)
* Role(Admin, User)
* Component
* Order

### Used libs and techs
* Spring Boot
* Spring Security
* HTML, CSS, Thymeleaf
* Liquibase
* Lombok
* Docker
* Database: PostgresSQL;

### Start Up

* Launch ComputerComponentsStore
* GoTo http://localhost:8090
